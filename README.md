## Directory Guide

### `/prompts`
Store your prompt files here for interacting with Claude or other AI systems.
- **templates/** - Reusable prompt patterns and templates
- **examples/** - Example prompts that worked well

### `/documents`
All generated documentation and notes.
- **generated/** - Documents automatically generated by Claude
- **notes/** - Your manual notes, observations, and findings

### `/plans`
Execution plans and task breakdowns.
- **execution/** - Numbered execution plans (plan-001.md, plan-002.md, etc.)
  - Each plan includes: date, request description, detailed steps, and results

## Naming Conventions

- Plans: `plan-001.md`, `plan-002.md`, `plan-003.md`, etc.
- Use lowercase and hyphens for file names
- Use descriptive names for clarity

## Getting Started

1. Add your prompts to the `prompts/` directory
2. Generated documents will be saved to `documents/generated/`
3. Execution plans will be tracked in `plans/execution/`
